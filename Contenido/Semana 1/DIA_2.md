# Introducción a Fiori
Es un sistema de diseño que reflejan una nueva forma de trabajo con las soluciones SAP. Fiori brinda una experiencia de usuario basada en roles que se pueden personalizar en todas las líneas de negocio, tareas y dispositivos. En este sentido, se habla de una nueva Experiencia de Usuario (UX: User Experience)., generando y brindando una navegación satisfactoria y eficaz dentro del sistema. 
## Fiori Design Guidelines 
Son principios de diseño establecidos por SAP para garantizar una experiencia de usuario coherente y eficiente en todas las aplicaciones Fiori.
Principios clave:
-Rol-Centric: Diseñadas para cumplir necesidades específicas de cada usuario.
-Responsivas: Adaptables a diferentes dispositivos y tamaños de pantalla.
-Coherentes: Todas las aplicaciones Fiori siguen un mismo estándar visual y funcional.
-Simplicidad: Interfaces limpias y fáciles de usar con reducción de elementos innecesarios.
-Experiencia del usuario intuitiva: Basada en interacciones naturales y amigables.
## SAP UI5 
SAPUI5 es un marco de desarrollo (framework) basado en JavaScript que SAP utiliza para crear aplicaciones web modernas y enriquecidas, especialmente diseñadas para integrarse con sistemas SAP como SAP S/4HANA, SAP Fiori, y otros servicios de SAP. Su propósito principal es proporcionar una experiencia de usuario (UX) fluida, interactiva y adaptativa en dispositivos móviles, tabletas y escritorios, independientemente de la plataforma o el navegador.
